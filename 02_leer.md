# Leer

```
Algoritmo leer
	Leer lista_de_variables
FinAlgoritmo
```

## Ejemplos

### Ejemplo 1
```
// leer numero entero

Leer(num)
```
### Ejemplo 2
```
// leer numero en decimal

Leer(num)
```
### Ejemplo 3
```
// leer cadena de texto

Leer(texto)
```
