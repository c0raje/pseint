# Asignar

```
Algoritmo asignar
	variable<-expresion
FinAlgoritmo
```

## Ejemplos

### Ejemplo 1
```
// asignar un valor entero a una variable

num <- 10
```
### Ejemplo 2
```
// asignar un valor decimal a una variable

num <- 3.14
```
### Ejemplo 3
```
// asignar un valor de texto a una variable

texto <- "Hola, mundo!"
```

### Descripcion

El comando "asignar" en PSeInt se utiliza para asignar un valor a una variable. Permite almacenar un dato especÃ­fico en una variable para su posterior uso en el programa.
